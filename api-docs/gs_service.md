## Описание пакета gs_service

## Описание
Данный пакет предоставляет набор типов сервисов и сообщений, необходимых для работы Geoscan Pioneer Max

## Состав пакета:
1. Типы сервисов:
    * Accel.srv
    * Altitude.srv
    * Event.srv
    * Gyro.srv
    * Info.srv
    * Led.srv
    * Live.srv
    * Log.srv
    * LpsPos.srv
    * LpsVel.srv
    * LpsYaw.srv
    * Orientation.srv
    * Position.srv
    * PositionGPS.srv
    * Range.srv
    * Time.srv
    * Yaw.srv
